Meu nome é Pedro Sabino, sou estudante do Cotemig e apaixonado por tecnologia, especialmente jogos. Grande parte dos meus projetos gira em torno desse tema, pois para mim é algo muito relaxante — não apenas jogar, mas também criar.

Tenho grande interesse por cibersegurança, backend, desenvolvimento de jogos e também filosofia. Me considero uma pessoa crítica, focada e com valores éticos bem definidos. Prezo pela honestidade, tanto no trabalho quanto na vida.

Atualmente, busco aprimorar minhas habilidades e transformar ideias em projetos que unam técnica e criatividade. Sempre estudando para me tornar um profissional mais refinado e uma pessoa melhor.

The Last Fall é um jogo desenvolvido em Java e inteiramente jogado pelo prompt de comando, ou seja, feito completamente em texto.

A ideia do projeto era treinar conceitos básicos de MySQL, aplicando operações CRUD em uma linguagem de back-end, além de explorar de forma prática a programação orientada a objetos. Esse foi o meu primeiro projeto tanto em Java quanto utilizando POO.

Uma breve descrição:
Quando o Criador abriu o Vazio, a humanidade foi condenada a enfrentar seus próprios monstros. Um guerreiro sem nome tentou selar a fenda e desapareceu. Séculos depois, o abismo ainda respira horrores — e um novo azarado cai em suas profundezas. Seu destino decidirá se o mundo conhecerá um novo herói... ou mais um demônio.

O jogo ainda está em desenvolvimento, mas já está disponível no meu GitHub para quem quiser acompanhar o progresso.

O cubo mágico, é um projeto bem simples feito em C#, ele simula um cubo mágico 2x2. Foi meu primeiro projeto feito em C#, e o primeiro projeto mais complexo que eu tentei fazer.

Esse projeto tem o intuito de reforçar o conhecimento em arrays, loops e uma introdução a métodos.

Ele já está pronto e disponível para qualquer um no meu git-hub.

Binary game, é um projeto inspirado no jogo binary game, o objetivo é realizar o cálculo de números binários e decimais, passando de decimal para binário e vice-versa.

Esse projeto foi feito com o objetivo de aprimorar o conhecimento em métodos, loops e continuidade.

Ele não está totalmente pronto, mas já está disponível no meu GitHub para quem quiser acompanhar o progresso.
